# 🎨 Art Gallery Management System

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![MySQL](https://img.shields.io/badge/MySQL-Database-orange.svg)
![Status](https://img.shields.io/badge/Status-Active-success.svg)

A web-based art gallery application developed using Python and MySQL. This project provides a robust database integration for efficiently managing artworks, artists, and gallery exhibitions.

## ✨ Features

* **Artwork Management:** Add, update, view, and delete art pieces.
* **Database Integration:** Secure and structured data storage using MySQL.
* **User-Friendly Interface:** Easy navigation for both gallery administrators and visitors.
* **Dynamic Queries:** Filter and search artworks by category, artist, or date.

## 🛠️ Technologies Used

* **Backend:** Python
* **Database:** MySQL
* **Frontend:** HTML / CSS / JavaScript (Eğer kullandıysanız bunları bırakın, kullanmadıysanız bu satırı silebilirsiniz)
